What is this?

Currently, this is the home of the Registry API proposal. The registry API is a minimal common API describing tools that we proposal for support by multiple registries like Dockstore and Agora for the purposes of exchange, indexing, and searching.

How to view

See the swagger editor to view our schema in progress.

If the current schema fails to validate, visit debugging

How to contribute changes

Currently, we are soliciting comments on this schema. Contributions can be made in one of several ways:

  1. Comment or debate in the disqus section below
  2. Make pull requests to the schema document
  3. Discuss in relevant Google Group
  4. Edit these documents which are written in Jekyll and hosted on Github Pages

We can also get inspiration for now from the ga4gh/schemas document.
